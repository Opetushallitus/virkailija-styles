Modal

  • stylesheet
src/modules/modal.css  

Only styles are provided, create functionality with JS.

Block the background with the Overlay component while a modal is open.

Usage

  • Toggle the overlay's and modal's visibility with .oph-overlay-is-visible and .oph-overlay-bg classes on .oph-overlay element. See the JS tab for an example.

  • Set overflow: hidden on body to hide the window's scroll bar while the modal is open

  • .oph-modal-container and .oph-overlay-is-visible's z-indexes may have to overridden depending on your app

Accessibility

  • .oph-modal-container is made focusable by setting tabindex=-1, has a role=dialog attribute and aria-labelledby linking to .oph-modal-title

  • .oph-modal has a role=document attribute for screen reader users to browse the contents more easily

  • .oph-modal-back-to-top-link for setting focus back to .oph-modal-container. The link is initially visually hidden, and displayed on focus.
