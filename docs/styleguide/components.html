Usage

For individual component stylesheets, import from the dist folder:

@import "oph-virkailija-style-guide/dist/alerts.css"

Source order is important; import Typography (dist/typography.css) and Colors (dist/colors.css) last if you want to use them to override other components' styles.

Import src/modules/variables.css if you want to use the defined CSS variables in your CSS files.

For individual component stylesheets, copy from the dist folder and add link tags:

<link href="{path/to/resources}/alerts.css" rel="stylesheet">
