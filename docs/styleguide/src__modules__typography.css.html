Typography

  • stylesheet
src/modules/typography.css  

Typography base

Set your app's or component's font family, size, color and line height with .oph-typography. Base font size is set to 95% of the browser's default font size.

Add the class as high in the element hierarchy as possible, preferably to html or the main element of your app.

Type scale

Adjust elements' font size with oph-h{1-6}. Overrides default size.

Rem units are used for sizing. Combine the classes with fluidly sized elements for best results.

Headings (and bolded text)

Style heading with .oph-bold and the font size classes. Font weight is set to 600.
